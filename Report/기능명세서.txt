<기능 명세서>
0. 최초 로그인
App을 받은 사용자들은 최초 로그인을 시도하며, 이 로그인 정보는 종합정보시스템과 연결된다. 즉 학생들은 별도의 회원 가입 없이 자신의 학번과 비밀번호를 이용해 로그인 할 수 있다.
로그인 화면은 한성대학교 로고와 ID, Password를 입력하는 Text 입력 창이 두 개 있다.
ID 입력 창에는 ‘학번’, Password 입력창에 ‘비밀번호’ 라는 이라는 불투명한 글씨가 써 있으며, Text창을 Touch 한 경우 키보드가 올라오고, ‘학번’, ’비밀번호’ 라고 쓰인 글씨는 사라지고 입력을 위한 커서가 활성화된다.
비밀번호는 사용자들에게 ***3처럼 가장 마지막으로 쓴 문자 외에는 별 문자(* )로 표시한다.
ID또는 PW가 잘못된 경우, “학번 또는 비밀번호가 맞지 않습니다” 라는 메시지를 ID텍스트박스 상단에 표시한다.

1. 메인 화면
로그인을 완료한 상태라면 앞으로 App을 실행 시 메인 화면이 나타난다.
메인 화면은 가운데 한성대학교 로고가 있고, 좌측 위에 흰색 三 모양의 메뉴 버튼이 있으며, 로고 아래로 두 가지 주요기능 버튼(캠퍼스맵, 시간표)이 있다.
메뉴 버튼을 클릭하면 Facebook처럼 슬라이드메뉴가 나오고, 왼쪽상단에 학번과 자기 이름(본인 정보)이 나오며, 여기서 Logout이 가능하다.
Logout 된 경우, 다시 로그인을 위한 0. 과정으로 되돌아간다.

2. 주요 기능 
1) Campus Map
캠퍼스맵의 첫 시작화면은 [타이틀/검색버튼], 학교 전체 건물 image (평면도)
타이틀은 한 줄로 되어있으며, 타이틀에서는 현재 보여주고 있는 위치명을 표시.
Ex) 우촌관 1층, 창의관 2층
또한 타이틀 오른쪽 끝에는 돋보기 모양의 검색 버튼이 있음.

?	검색 버튼을 사용한 경우
돋보기 버튼을 누른 경우 타이틀 Text가 사라지며 검색어를 입력할 수 있는 커서가 활성화됨.
이 때 (커서가 활성화 되었을 때), 학교 전체 건물image는 비활성화, 키보드 활성화.
만약 검색창, 키보드 외 비활성화 된 부분을 클릭한 경우 또는 뒤로가기 버튼을 누른 경우에는.
키보드 사라짐. 이전 화면으로 돌아감.
검색창에 검색어를 입력하면, 입력받는 도중에 관련 단어(장소)들이 리스트로 표시됨.
나열된 리스트들은 장소의 이름과 위치를 짧은 Text로 표시
Ex) 예~
예술대학 사무실		창의관 XXX호
예비군연대 		탐구관 XXX호
리스트를 선택하면 상세정보가 Dialog형태로 나옴

?	학교 전체 image에서 건물을 클릭한 경우 (평면도를 사용한 경우)
건물을 클릭한 경우, 그 건물의 층별 평면도가 나옴.
왕래가 잦은 출입구가 있는 층을 먼저 보여줄 것임.
전체적인 Layout은 [타이틀/검색Button] / 건물의 층 image / 층 이동 Button / 회전 Button.
타이틀의 오른쪽 끝부분에 위치한 검색 버튼을 누른 경우, 키보드 활성화.
어디서든 검색 가능하도록 검색창은 항상 활성화 가능하도록 함. (=검색버튼 사용)
건물의 층별 image가 나와서 각 층에 어떤 강의실/사무실이 있는지 알 수 있음.
image의 방을 클릭한 경우, 방에 대한 상세정보가 나옴 (xx과 사무실, 전화번호, Fax....)
맨 아래 층 버튼을 선택하거나 평면도에서 좌우 슬라이드를 했을 때, 다른 층으로 이동.
정가운데엔 회전 버튼이 있어서 누르면 평면도를 180도 회전.
방을 클릭하면 상세정보가 Dialog형태로 나옴
핸드폰의 뒤로가기 버튼을 누른 경우, Campus Map 메인화면으로 돌아간다. (전체 건물 평면도)
 

2) 시간표
시간표 기능은 종합정보시스템에 로그인 된 시간표 데이터를 가져와서 우리가 꾸민 Layout으로 보여 줄 것이며, 사용자들은 종합정보시스템의 시간표 정보 수정 및 일정 추가가 가능하다.
시간표 기능의 첫 화면은 상단 타이틀과 일일시간표가 보여지도록 구성되어 있다.
상단 타이틀은 [메뉴버튼 / 요일표시 or ‘시간표’ / 화면전환버튼 / 추가버튼]으로 구성되어 있다. 먼저 메뉴 버튼으로는 전체적인 시간표 기능의 옵션을 관리한다.
- 시간표 기능의 기본 화면을 일일시간표/전체시간표 로 할 것인지 설정하는 토글 (좌/우)
- 시간표를 종합정보시스템으로부터 새로 받아오는 버튼 (초기화)
(기존 시간표가 제거되고 새로 시간표를 받아옵니다 [Yes/No] )
- 시간표를 넣을 요일 범위 설정 (Ex 일월화수목금토 or 월화수목금토) 
상단 Bar의 화면 전환버튼은 일일시간표 Mode와 전체시간표 Mode를 변경할 수 있다.
하지만 화면 전환버튼은 기본 화면을 다른 Mode로 설정하지는 못한다. 기본 화면 설정은 메뉴버튼에서 할 수 있다.
또한 추가 버튼을 누를 경우, 종합정보시스템에서 받아온 일정 외에 새로운 일정을 추가한다.
일정 추가는 요일/시간/과목명/강의실/교수님명/전화/E-mail/홈페이지 의 내용을 작성할 수 있으며, 일정 상세정보의 Edit 기능과 같은 형식을 가진다.
시간표 기능은 일일시간표와 전체시간표 두 가지 Layout으로 구성되어 있다.
일일시간표와 전체시간표 두 가지 Mode의 차이점은,
- 일일시간표 : 그날의 시간표 만을 보여주며 Simple함을 최대한 강조한 기능.
- 전체시간표 : 우리에게 익숙한 표의 형태로, 한주간의 전체적인 일정을 한눈에 보여준다.
두 시간표 모두 기본적으로 보이는 정보는 해당 일정의 과목명과 강의실이며, 추가적인 정보를 보고 싶은 경우, 일정을 Touch 하면 상세정보가 Dialog로 나온다. (팝업 창 형식, 창 주변 불투명)
일정 상세정보 Dialog의 경우, 상단에는 과목명과 Edit 버튼이 있다. Dialog의 내용으로는 강의실, 교수님 명(연구실 위치), 교수님 전화번호, E-mail, 과목 홈페이지가 나온다. 또한 Dialog에서의 Edit 버튼을 누른 경우, 현재 내가 보고 있는 일정을 수정할 수 있는데, 일정 추가 기능과 마찬가지로 요일/시간/과목명/강의실/교수님명/전화/E-mail/홈페이지 라는 형식을 가지되, 종합정보시스템에서 받아왔던 정보들이 미리 작성되어있다. 이것들을 수정하거나 교수님의 연락처, 과목 홈페이지 등 본인이 원하는 정보를 추가 작성 할 수 있다.
