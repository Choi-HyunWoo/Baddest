화면의 구성

View		: 화면을 구성하는 모든 구성 요소 (화면의 일정 공간을 할당받는 것들)
View group 	: 화면을 놓는 공간 : View group
Widget 		: 뷰 중에서 일반적인 컨트롤의 역할을 하고 있는 것 (버튼, 텍스트 등)
Layout 		: 뷰를 어떻게 배치할 것인지 결정하는 것


액티비티 : 화면
액티비티가 만들어질 때 oncreate() 호출
